Upgrades Next.js and Nextra.

- **Next.js v16.x is now required**. v15.x is no longer supported.
- `nextra` version should be > `4.6.1`

| Package | From | To |
|---------|------|-----|
| `next` | 15.x | 16.1.6 |
| `@next/mdx` | 15.x | 16.1.6 |
| `nextra` | 4.x | 4.6.1 |.
