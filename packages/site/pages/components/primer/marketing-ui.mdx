Marketing React components can be used in Markdown files by importing them from `@primer/react-brand`.

[Refer to the Primer Brand documentation](https://primer.style/brand) for more information on which components are available and how to use them.

## Using Primer Brand in Markdown files

1. Import a component from `@primer/react-brand`:

```js
import {Button} from '@primer/react-brand'
```

2. Use the component in your Markdown file, but ensure that you wrap the component in a `div` with the `custom-component` class. This will ensure that article formatting isn't applied to the custom component.

```jsx
<div class="custom-component">
  <ButtonGroup>
    <Button>Primary</Button>
    <Button>Secondary</Button>
  </ButtonGroup>
</div>
```

Will render:

<br />

<div class="custom-component">
  <ButtonGroup>
    <Button>Primary</Button>
    <Button>Secondary</Button>
  </ButtonGroup>
</div>

### Layout

Use [Stack](https://primer.style/brand/components/Stack), [Grid](https://primer.style/brand/components/Grid) and [Box](https://primer.style/brand/components/Box) components to create custom layouts.
